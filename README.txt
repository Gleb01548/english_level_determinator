Цель проекта: создать систему для определения уровня языка фильма. 
Источник данных: название и оценки фильмов puzzle-movies.com, к ним субтитры. 

Сформирован датасет из 3.5 тыс. фильмов. Релизована LSTM сеть, 
которая не работает (переобучается со 2-3 эпохи). Необходимо попробовать другие подходы.
